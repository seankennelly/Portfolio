Gazetteer Project

Project Description

The brief for this project was to create an interactive gazetteer-style map web application, populated by geographical data pulled from APIs. The app needed to be responsive and pull data from a variety of sources to create an engaging end-user experience. Due to its highly interactive nature I focused on a minimalist design, encouraging users to explore the available controls to discover functionality.

Moonrise Maternity Website

Project Description

This site was built according to client specifications, and using content produced by the client themselves. The key challenge of the site was to lay out a large amount of text content without bulky paragraphs. I used Bootstrap components (accordions, carousels etc) to allow users to interact with an otherwise static page. I discussed and confirmed colour and design conventions with the client, and created a business logo for them. I also host the site, and set up a business email address for them.

The Witchcraft & Vagrancy Act Website

Project Description

The client presented comprehensive designs based around a desktop layout. I adapted them to be mobile-first with cross-browser compatibility. Communication with the client was paramount, ensuring that the smaller-screen designs fit with their original vision. The site was principally built with Bootstrap and Flexbox. Initially, I found the high-quality images to be slowing the site considerably, but learned to minimise file sizes during this project. I host and administer this site for the client, and will continue to update and maintain it.
